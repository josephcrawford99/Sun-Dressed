# Clothing for Weather
The following is a spec sheet for an app I would like to build for iphone and android. The app suggests clothing based on the weather.

## MVP Features
- [ ] User can specify their location
- [ ] User can see the weather forcast for the day in the morning, afternoon, and evening
- [ ] User sees a list of clothing suggestions for each time of day

## Further features
- [ ] User can rate a suggestion from the app to improve the futuresuggestions (most important). Too hot or too cold, sunny, windy, rainy, etc.
- [ ] User can say what clothes they have available
- [ ] User can specify their clothing preferences
- [ ] User can be suggested clothing to buy
- [ ] App developer/manager can manage database of clothes and their properties
- [ ] User can see weather forecast for multiple days along with clothing suggestions
- [ ] User can see clothing suggestions for a range of days for a trip
